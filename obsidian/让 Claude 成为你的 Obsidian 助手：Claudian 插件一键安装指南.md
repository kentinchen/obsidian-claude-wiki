原创 悟鸣 *2026年2月28日 07:46*

最近发现，身边很多人使用 Obsidian，并通过 Claudian 插件集成 Claude Code。

有朋友问我：Claudian 是什么？怎么安装？有什么好处？

![图片](https://mmbiz.qpic.cn/mmbiz_png/d5FwhibzVibEEXoDR0O0n0Xs49YWcnjteyH9mzxaWTYfFv29SJHOnYvbbcSSXeesqqXGdL5d8qK1SJzBpse2OtEsJVHg96gTqnvSKuMajFUibc/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)

Claudian 是基于 Claude SDK 的 Obsidian 插件。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/d5FwhibzVibEGQ8wP9EnPx8ug5w1tleDGnV5CLcUz63tAib3dxEibxjN6yONvYsMIXYs1aPA7jIAb0fQNV2dlyJPblRGxg6YAgTC2NuRPZYs4os/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

不需要切换工具，直接在笔记中与 Claude 对话，并且能读取笔记内容作为背景知识。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

Claudian 实际体验下来有几个优点：

- 界面美观、加载快速
- 支持和当前文档对话
- 支持查看对话历史
- 当前打开的文件和圈选文本自动添加到右下角的上下文中
- 支持添加外部上下文
- 支持模型切换、设置思考长度、YOLO 和 Safe 模式切换等

## 手动安装（4步）

打开官方 GitHub 一看，安装步骤让我有点傻眼。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

**第一步** ：从 latest release 下载 `main.js` 、 `manifest.json` 、 `styles.css` 三个文件。

**第二步** ：在 vault 目录中创建插件子目录

```
/path/to/vault/.obsidian/plugins/claudian/
```
![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

**第三步** ：将上述 3 个文件复制进去

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

**第四步** ：在 Settings → Community plugins → 启用 "Claudian"

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

点击左下角的机器人图标，就可以在侧边栏与 Claude Code 对话了。
